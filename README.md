# Magnit Kosmetik Testing Project

## 📌 Описание
Данный проект содержит результаты тестирования веб/приложения "Магнит Косметик".

## 📁 Структура проекта
- docs — тестовая документация
- test-cases — тест-кейсы
- bug-reports — найденные дефекты
- screenshots — скриншоты приложения
- reports — итоговый отчёт

## 🧪 Что было выполнено
- Анализ требований
- Составление тест-кейсов
- Функциональное тестирование
- Регрессионное тестирование
- Заведение баг-репортов
- Фиксация результатов

## 📸 Скриншоты
Приложены реальные скриншоты тестируемого приложения.

## 📊 Итог
Приложение протестировано, найдены и задокументированы дефекты.
